Токен VK (можно вставить вручную)
Ручной токен через vkhost.github.io: выбери Kate Mobile. В «Настройки» на vkhost отмечай нужные права (аудио, офлайн) — там как раз галочки. Затем войди: сначала может открыться VK ID («Продолжить как …») — нажми продолжить; на странице id.vk.com со списком прав нет отдельных галочек по строкам, это просто перечень доступа приложения; внизу страницы найди кнопку «Разрешить» (при узком окне — прокрути вниз или разверни браузер на весь экран). После перехода скопируй из адреса фрагмент от access_token= до &expires_in.
Обход через Chrome (Selenium)
Только если VK режет audio.*: отдельное окно Chrome, вход в VK, профиль в %LOCALAPPDATA%\Flow\vk_chrome_profile (имя папки историческое). По умолчанию выключено — поиск и импорт не запускают браузер сами.
Не настроен
Нужен OAuth token для чтения плейлистов
Актуально: OAuth Device Flow 1. Установи Python-библиотеку: pip install -U yandex-music 2. Запусти короткий скрипт из документации, открой ссылку и введи код. 3. Скопируй access_token сюда. NexoryND берёт из Яндекса только названия и артистов, потом ищет треки обычным поиском. Запасной вариант: браузерная OAuth-ссылка ниже. После входа скопируй access_token из URL.
