# Как подключить ВКонтакте во Flow — токен

*Скопируй блоки ниже в [@Telegraph](https://t.me/telegraph) — получится страница как у гайда по Яндексу.*

Чтобы во **Flow** работали поиск по аудио ВКонтакте и импорт плейлистов, приложению нужен **access token** с доступом к аудио.

### Шаг 1. Открой страницу получения токена

Перейди на https://vkhost.github.io/

### Шаг 2. Выбери Kate Mobile

В списке приложений выбери **Kate Mobile**.

### Шаг 3. Отметь права, особенно «Аудио»

На vkhost отметь нужные права. Для музыки критично право **«Аудио»**.

### Шаг 4. Войди в VK и разреши доступ

Может открыться **VK ID** — подтверди вход.

На странице id.vk.com со списком прав **нет отдельных галочек по строкам** — это перечень доступа приложения. Внизу нажми **«Разрешить»** (при узком окне прокрути вниз).

### Шаг 5. Скопируй токен из адреса

В адресной строке будет фрагмент:

`...#access_token=ТОКЕН&expires_in=...`

Скопируй значение от `access_token=` до `&expires_in`.

### Шаг 6. Вставь токен во Flow

Настройки → Аккаунты → Источники или «Авторизация VK» → вставь токен → сохрани → «Проверить токен».

### Если что-то не так

- Код **5** — токен недействителен, получи новый.
- Код **6** — слишком много запросов, подожди и повтори проверку.
- Код **15** — проверь тип токена и права (Kate + аудио).
- Если VK режет `audio.*`, в настройках VK есть опция **обхода через Chrome (Selenium)** — только при необходимости.
