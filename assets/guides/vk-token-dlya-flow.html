Как подключить ВКонтакте в Nexory — токен

Гайд для десктопного Nexory · поиск и импорт VK

Чтобы в Nexory работали поиск по аудио ВКонтакте и импорт плейлистов, приложению нужен access token с доступом к аудио. Ниже — самый распространённый путь через сервис vkhost и приложение Kate Mobile.

Шаг 1. Открой страницу получения токена

Перейди на https://vkhost.github.io/.

Шаг 2. Выбери приложение Kate Mobile

В списке приложений выбери Kate Mobile (или эквивалент с полноценным доступом к аудио — как подсказывает vkhost).

Шаг 3. Отметь права, особенно «Аудио»

На vkhost отметь нужные права. Для музыки критично право «Аудио» (и связанные пункты, если они есть в интерфейсе).

Шаг 4. Войди в VK и разреши доступ

Нажми кнопку входа / получения токена. Может открыться VK ID («Продолжить как …») — подтверди.

На странице id.vk.com со списком прав нет отдельных галочек по каждой строке: это перечень того, что запрашивает приложение. Пролистай вниз и нажми «Разрешить». Если кнопки не видно — разверни окно на весь экран или прокрути страницу вниз.

Шаг 5. Скопируй токен из адреса

После успешного входа браузер перейдёт на redirect-страницу, а в адресной строке будет фрагмент вида:

...#access_token=СЮДА_ДЛИННЫЙ_ТОКЕН&expires_in=...

Скопируй значение от access_token= до следующего & (до expires_in и т.п.). Это и есть токен.

Шаг 6. Вставь токен в Nexory

Открой Nexory → Настройки → раздел Аккаунты → блок Источники или секция «Авторизация VK».

Вставь токен в поле, нажми сохранить (галочка), затем при необходимости — «Проверить токен».

Если проверка ругается на audio.* или коды VK

  • Код 5 — токен недействителен, получи новый.
  • Код 6 — слишком много запросов: подожди 30–60 секунд и проверь снова.
  • Код 15 — часто связано с типом токена: для аудио нужен подходящий токен (Kate и корректные права).
  • Если VK «режет» официальные методы audio.* для части аккаунтов, в Nexory есть опция обхода через Chrome (Selenium) в настройках VK — только если реально нужно; по умолчанию браузер сам не открывается.
